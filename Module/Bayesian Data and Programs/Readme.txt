The programs in this folder are for the class on 
Bayesian Analyses Using SAS

You need to open and submit the program STBAY00d01.sas before running any of the programs.  
The program STBAY00d01.sas generates all of the SAS data sets for this course 
and stores them in the SASUSER library.
 
The demonstration programs, solutions to the exercises, and the appendix programs are also provided.
